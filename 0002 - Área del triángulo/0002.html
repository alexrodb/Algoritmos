0002 - Área de un triángulo

Encontrar el modo para calcular e imprimir el área de un triángulo.

Pseudocódigo:

      Algoritmo areaTriangulo
        Definir b,h,A Como Real
        Escribir 'Digite la base del triángulo'
        Leer b
        Escribir 'Digite la altura del triángulo'
        Leer h
        A <- (b*h)/2
        Escribir 'El área del triángulo es: ',A
      FinAlgoritmo

Complejidad:

Mismos números de pasos, siempre recibimos los mismos datos, es constante.
