0001 - Mínimo de 3 números

Encontrar el mínimo de 3 números, no importa si los números son negativos o positivos, decimales o enteros, irracionales o trascendentes.

Pseudocódigo:

    Mínimo de 3 números (a, b, c)
    Si a menor que b entonces 
    min2 = a
    si no
    min2 = b
    si min2 menor que c entonces
    min 3 = min2
    si no 
    min3 = c

Complejidad:

Mismos números de pasos, siempre recibimos los mismos datos, es constante.
